# 2019_SmartWatch


Overwiew:
  ePaper display 1,54' supported by Android application. Communication rely on Bluetooth module 2.0.
  Project 
 
Description:
  Microcontroler communicates with An

SmartWatch:

Hardware:
- STM32F4,
- ePaper display 1,54",
- Bluettoth module 2.0, HC-06.

Setup:

  STM32:

  - Screen:
      CS >> PB6
      RST >> PA9
      BUSY >> PA8
      DC >> PC7
      DIN >> PA7
      CLK >> PA5
      GND >> GND
      3.3V >> Vcc

  - Bluetooth
      RxD >> PB10
      TxD >> PB11
