# 2019_SmartWatch

Wyświetlacz ePaper 1,54' połączony za pomocą Bluetooth z aplikacją na telefony z systemem Android.

SmartWatch:

Wykorzystano:
- mikrokontroler STM32F4,
- wyświetlacz w technologii ePaper 1,54",
- modułe Bluettoth wersja 2.0 HC-06.


Setup:

  STM32:

  - Screen:
      CS >> PB6
      RST >> PA9
      BUSY >> PA8
      DC >> PC7
      DIN >> PA7
      CLK >> PA5
      GND >> GND
      3.3V >> Vcc

  - Bluetooth
      RxD >> PB10
      TxD >> PB11
